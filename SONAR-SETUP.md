Sonar configurations


1. Add the sonar configurations like sonar server URL and token of the sonarqube to connect to DB in POM.XML
2. Before adding the sonar token you have to create the token for sonarqube for security purpose.
3. To create the token for sonarqube, you have to login to sonarqube and click on your profile and then click on my account and then click on security and then click on generate token and give the name of the token and then click on generate and copy the token and paste it in POM.XML
   a) Go to sonarqube and login with your credentials. Select Adminstration -> Security -> Users -> Select your user and click on Generate Tokens. Give the name of the token and click on Generate. Copy the token and paste it in POM.XML.
